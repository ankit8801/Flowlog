# Dashboard Package

This folder contains the Next.js dashboard for Focus Tracker.

For complete project documentation (architecture, extension setup, scripts, deployment, and troubleshooting), see the repository root README:

- ../README.md

Quick local start from this folder:

```bash
npm install
npm run dev
```

Dashboard URL:

- http://localhost:3000

Useful scripts in this package:

- npm run dev
- npm run build
- npm run start
- npm run lint
- npm run zip:extension
- npm run zip:watch
